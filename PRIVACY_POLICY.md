# Privacy Policy

# NexGen App Vault Scanner
**Developed by NexGen Dev Studio**

**Effective Date:** July 2026

---

## Introduction

Welcome to NexGen App Vault Scanner.

Your privacy is important to us. NexGen App Vault Scanner is designed to help Android users better understand application permissions and privacy-related information while maintaining complete transparency about how the application operates.

Our mission is to provide users with clear, informative, and privacy-focused insights without collecting or sharing personal data.

---

## Application Features

NexGen App Vault Scanner includes the following features:

- Scan installed Android applications
- Analyze sensitive permissions
- Permission-based Privacy Risk Score
- Safe, Medium, and High Risk categories
- Detailed application information
- Package name information
- Permission breakdown
- Scan history
- Fast offline scanning
- Premium modern user interface
- Multiple navigation sections for easy access
- Smooth and user-friendly experience

---

## Purpose of the Application

This application is created to improve privacy awareness.

The information displayed is intended to help users understand the permissions requested by installed applications and make informed decisions regarding their privacy.

---

## How Privacy Risk Scores Are Generated

Privacy Risk Scores are automatically calculated using the permissions requested by installed applications together with publicly available Android application metadata.

These scores are provided only as an informational privacy indicator.

They do **not** determine whether an application is:

- Malware
- Virus
- Harmful
- Dangerous
- Fraudulent
- Unsafe
- Infected
- Spyware

---

## Third-Party Applications

NexGen App Vault Scanner does not accuse, certify, rate, or officially classify any third-party application as malicious or unsafe.

All information displayed inside the application is generated automatically through permission analysis.

The application does not modify, inspect private user content, or interfere with third-party applications.

Users should always make their own decisions before installing, removing, or trusting any application.
